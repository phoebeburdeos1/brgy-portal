# Barangay Bonbon System

Next.js + Supabase portal for the barangay: public booking, captain status, announcements, and admin dashboard with email (Resend).

## Setup and run

See **[next-app/README.md](next-app/README.md)** for Supabase setup, env vars, and `npm run dev`.

## Project structure
